REST Api Documentation (GameLib)

/ = Documentation


GET /users = Show all users

GET /user/:id = Show one specific user

POST /user = Post a user

POST /user/login = Checks if user send the right data

DEL /user/:id = Delete an user

PUT /user/:id = Update email or password of user


GET /game = Show all games

GET /game/:id = Show one specific game

GET /game/saved: :id = Show all games saved by a user

POST /game = Post a game

DEL /game/:id = Delete a game
